Conexión hardcodeada

Esta versión usa la misma lógica de acceso del ejemplo proporcionado: autenticación MSAL, token de Microsoft Graph y acceso al archivo mediante un identificador compartido EXCEL_FILE_ID.

Nota: ya no se usa configuración local del navegador. Si falla la conexión, el sistema muestra el error y no carga información demo.
