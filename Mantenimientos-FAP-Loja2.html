Parámetros de Integración MS Graph

La aplicación consume de forma síncrona el SDK de Microsoft Graph y MSAL para conectarse de manera segura a la cuenta corporativa del FIAS y actualizar las tablas del libro maestro.

Nota técnica: Se implementó la codificación Base64URL segura para tokens compartidos de SharePoint (`u!`).
Estructuras requeridas en Excel
tblVehiculos tblIntervenciones tblDetalle tblAuditoria

Asegúrese de que el archivo compartido posea estas cuatro tablas con nombres canónicos estructurados mediante formato de tabla de Excel.
